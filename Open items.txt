5.What is OpenCV?

    OpenCV (Open Source Computer Vision Library) is an open-source library used for:

    Reading images
    Reading videos
    Accessing webcams
    Processing images
    Detecting objects
    Drawing shapes
    AI preprocessing
    Computer Vision applications

    Think of OpenCV as the eyes of your application.

    Without OpenCV:
    Your AI Model
     ↑
    No Image


    With OpenCV:
    Camera
    │
    ▼
    OpenCV
    │
    ▼
    Image Frame
    │
    ▼
    MediaPipe / AI Model

->  Application flow is:
    Webcam
    │
    ▼
    OpenCV
    │
    ▼
    Video Frame
    │
    ▼
    MediaPipe
    │
    ▼
    21 Hand Landmarks
    │
    ▼
    Random Forest Model
    │
    ▼
    Gesture Name
    │
    ▼
    Text To Speech    

->  OpenCV is responsible only for:

    Opening webcam
    Capturing frames
    Drawing text
    Showing processed images

-> What is MediaPipe?
    MediaPipe is an open-source framework developed by Google for building real-time AI pipelines.

    It can detect:
    ✋ Hands
    😀 Face
    🧍 Human Pose
    👁 Iris
    🤳 Selfie Segmentation
    📦 Object Detection (with other models)

    Think of MediaPipe as a real-time AI detector.
